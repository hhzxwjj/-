## 8.1 规模估算

### 8.1.1 代码规模

| 项目 | 规模 |
|------|------|

| HTML 模板 | 14 个模板文件，约 2,500 行 |
| CSS 样式 | 约 800 行 |
| JavaScript | 约 100 行 + 模板内联脚本 |

```bash
# 查看文档提交历史
git log --oneline docs/

# 示例输出
feat(docs): 编写可行性分析与软件开发计划
docs(requirements): 完善需求分析与建模文档，补充ER图和数据字典
feat(design): 完成概要设计和详细设计文档
docs(test): 补充测试策略和测试用例
docs(deploy): 编写用户手册和部署文档
```
