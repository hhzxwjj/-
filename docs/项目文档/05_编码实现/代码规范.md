## 1. 命名规范

### 1.1 Python 命名

| 类型 | 规范 | 示例 | 说明 |
|------|------|------|------|
| 模块/包 | 小写 + 下划线 | `database.py`, `main.py` | 简短、有意义 |
| 类 | 大驼峰 | `DatabaseConnection`, `RateLimiter` | 首字母大写 |
| 函数/方法 | 小写 + 下划线 | `get_db_connection()`, `hash_password()` | 动词开头 |
| 变量 | 小写 + 下划线 | `user_id`, `course_name` | 有意义的名词 |
| 常量 | 全大写 + 下划线 | `MAX_REQUESTS`, `PHONE_PATTERN` | 模块级常量 |
| 私有属性 | 单下划线前缀 | `_secret_key`, `_get_mysql_conn()` | 内部使用 |
| 受保护属性 | 双下划线前缀 | `__enter__`, `__exit__` | Python 特殊方法 |

**示例**：

```python
# ✅ 正确
PHONE_PATTERN = re.compile(r'^1[3-9]\d{9}$')

def get_db_connection():
    """获取数据库连接"""
    pass

class DatabaseConnection:
    """数据库连接上下文管理器"""
    
    def __enter__(self):
        self.conn = get_db_connection()
        return self.conn

# ❌ 错误
phonepattern = re.compile(...)     # 不使用驼峰
def getDBConnection():              # 不使用驼峰
    pass
class databaseConnection:           # 类名小写
    pass
```

### 1.2 数据库命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 表名 | 小写 + 下划线 | `users`, `chat_pins` |
| 字段名 | 小写 + 下划线 | `user_id`, `created_at` |
| 主键 | `id` | `id INTEGER PRIMARY KEY` |
| 外键 | `表名_id` | `user_id`, `course_id` |
| 索引 | `idx_字段名` | `idx_user_id` |
| 唯一约束 | `uk_字段名` | `uk_username` |

### 1.3 路由/URL 命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 页面路由 | 小写 + 连字符 | `/user-center`, `/complete-profile` |
| API 路由 | 小写 + 下划线 | `/api/contacts`, `/api/toggle_pin` |
| 资源路由 | 名词复数 | `/courses`, `/messages` |
| 动作路由 | 动词 | `/send_message`, `/move_student` |

> 注：当前项目中页面路由使用下划线（历史原因），新路由建议统一。

## 2. 代码格式

### 2.1 缩进与空白

```python
# ✅ 正确：4 空格缩进，不使用 Tab
def example():
    if condition:
        do_something()
    else:
        do_other()

# ✅ 正确：运算符两侧加空格
x = a + b
```

```python
# ✅ 正确：括号内换行
result = some_function(
    arg1,
    arg2,
    arg3
)
```

## 3. 注释规范

### 3.1 文档字符串（Docstring）

**所有模块、类、函数都必须有文档字符串**。

```python
"""
模块级文档字符串

说明模块的用途、主要功能、使用示例。
"""


def hash_password(password: str) -> str:
    """使用 PBKDF2-HMAC-SHA256 对密码进行哈希加密。
    
    生成随机 salt，使用 10 万次迭代，返回 salt$hash 格式字符串。
    
    Args:
        password: 明文密码
        
    Returns:
        哈希后的密码字符串，格式为 "salt$hash"
        
    Example:
        >>> hash_password('123456')
        'a3f5c2...$b7e9d1...'
    """
    salt = secrets.token_hex(16)
    key = hashlib.pbkdf2_hmac('sha256', password.encode('utf-8'),
                               salt.encode('utf-8'), 100000)
    return f'{salt}${key.hex()}'


class DatabaseConnection:
    """数据库连接上下文管理器。
    
    自动处理事务提交/回滚和连接关闭。支持 SQLite 和 MySQL。
    
    Usage:
        with DatabaseConnection() as conn:
            cursor = conn.execute("SELECT * FROM users")
            rows = cursor.fetchall()
    """
```

```python
# ✅ 正确：注释说明"为什么"而非"做什么"
# 本地开发使用 HTTP，生产环境必须改为 True

# ✅ 正确：复杂逻辑加注释
# 根据签到状态变化更新课时统计
if old_status != 'present' and new_status == 'present':

# ❌ 错误：注释重复代码

# ❌ 错误：注释与代码不一致
# 设置为 True（实际代码是 False）
SESSION_COOKIE_SECURE = False
```

## 6. 错误处理规范

### 6.1 异常处理

```python
# ✅ 正确：捕获具体异常，不裸捕获
try:
    conn.execute('BEGIN TRANSACTION')
    # ...
    conn.commit()
except sqlite3.IntegrityError as e:
    conn.rollback()
    app.logger.error(f'数据库约束错误: {e}')
    return error_response('数据冲突，请重试')
except Exception as e:
    conn.rollback()
    app.logger.error(f'未知错误: {e}', exc_info=True)
    return error_response('操作失败，请稍后重试')
finally:
    conn.close()

# ❌ 错误：裸捕获，吞掉所有异常
try:
    do_something()
except:           # ❌ 不指定异常类型
    pass          # ❌ 什么都不做
```

### 6.2 日志记录

```python
# ✅ 正确：使用 logging 而非 print
import logging

app.logger.info(f'用户 {username} 登录成功')
app.logger.warning(f'验证码不匹配: 输入={input_code}')
app.logger.error(f'数据库连接失败: {e}', exc_info=True)

# ❌ 错误：使用 print
print(f'用户 {username} 登录成功')   # ❌
```

## 7. 数据库操作规范

### 7.1 SQL 编写

```python
# ✅ 正确：使用参数化查询
conn.execute('SELECT * FROM users WHERE id = ?', (user_id,))

# ✅ 正确：SQL 关键字大写（可选但推荐）
conn.execute('''
    SELECT id, name, phone
    FROM users
    WHERE role = ? AND created_at > ?
    ORDER BY created_at DESC
''', ('user', '2026-01-01'))

# ❌ 错误：字符串拼接 SQL
# conn.execute(f"SELECT * FROM users WHERE id = {user_id}")   # ❌ SQL 注入风险
```

### 7.2 事务管理

```python
# ✅ 正确：使用上下文管理器
with DatabaseConnection() as conn:
    conn.execute('INSERT INTO users (...) VALUES (...)')
    # 自动提交/回滚

# ✅ 正确：手动事务
try:
    conn.execute('BEGIN TRANSACTION')
    conn.execute('UPDATE courses SET current_students = current_students + 1')
    conn.execute('INSERT INTO enrollments (...)')
    conn.commit()
except Exception:
    conn.rollback()
    raise
finally:
    conn.close()
```

### 7.3 连接关闭

```python
# ✅ 正确：确保连接关闭
def get_user(user_id):
    conn = get_db_connection()
    try:
        user = conn.execute('SELECT * FROM users WHERE id = ?', (user_id,)).fetchone()
        return user
    finally:
        conn.close()
```
