## 4. 代码组织原则

### 4.1 当前结构特点

**优点**：
- 单文件路由（`main.py`）便于快速开发和调试
- 模板与静态资源分离，职责清晰
- 数据库操作集中管理

**待改进**：
- `main.py` 1,740 行过大，应按功能拆分为多个 Blueprint
- `services/` 和 `utils/` 目录为空，未充分分层
- 缺少统一的配置管理文件（`config.py`）

### 4.2 建议重构后的结构

```

├── __init__.py              # 应用工厂模式
```

## 5. 关键文件职责

### 5.1 `run.py` — 启动入口

```python
#!/usr/bin/env python3
"""启动书法培训班管理系统"""

import os
from app import app, socketio
from app.models.database import init_db
from app.routes.main import init_admin

if __name__ == '__main__':
    init_db()           # 初始化数据库表
    init_admin()        # 初始化管理员账号
    
    # 确保目录存在
    os.makedirs('templates', exist_ok=True)
    os.makedirs('static', exist_ok=True)
    
    # 读取配置
    debug_mode = os.environ.get('FLASK_DEBUG', 'false').lower() == 'true'
    host = os.environ.get('FLASK_HOST', '0.0.0.0')
    port = int(os.environ.get('FLASK_PORT', 5000))
    
    # 启动服务
    socketio.run(app, debug=debug_mode, host=host, port=port, use_reloader=False)
```

### 5.2 `app/__init__.py` — 应用初始化

```python
from flask import Flask
from flask_socketio import SocketIO
import os
import secrets

app = Flask(__name__)

# SECRET_KEY 配置
_secret_key = os.environ.get('SECRET_KEY')
if not _secret_key:
    _secret_key = secrets.token_hex(32)
    print('[警告] SECRET_KEY 未设置，已自动生成临时密钥')
app.config['SECRET_KEY'] = _secret_key

# Session Cookie 安全配置
app.config.update(
    SESSION_COOKIE_SECURE=False,
    SESSION_COOKIE_HTTPONLY=True,
    SESSION_COOKIE_SAMESITE='Lax',
    PERMANENT_SESSION_LIFETIME=timedelta(hours=2)
)

# 静态资源路径
base_dir = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
app.static_folder = os.path.join(base_dir, 'static')
app.template_folder = os.path.join(base_dir, 'templates')

# SocketIO 初始化
socketio = SocketIO(app)

# 导入路由
from app.routes.main import *
```
