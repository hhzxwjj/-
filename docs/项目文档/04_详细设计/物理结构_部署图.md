## 2. 开发环境部署

### 2.1 环境要求

| 组件 | 版本 | 说明 |
|------|------|------|
| Python | 3.8+ | 运行环境 |
| SQLite | 内置 | 开发数据库 |
| pip | 最新 | 包管理 |

### 2.2 部署步骤

```bash
# 1. 克隆代码
git clone https://github.com/hhzxwjj/BaoMing.git
cd BaoMing

# 2. 创建虚拟环境
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
# .venv\Scripts\activate       # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动服务
python run.py

# 5. 访问
# http://localhost:5000
```

### 2.3 开发环境配置

```bash
# Windows 环境变量（PowerShell）
$env:FLASK_DEBUG="true"
$env:FLASK_HOST="0.0.0.0"
$env:FLASK_PORT="5000"

# Linux/Mac 环境变量（Bash）
export FLASK_DEBUG=true
export FLASK_HOST=0.0.0.0
export FLASK_PORT=5000
```

### 2.4 开发环境特点

| 特性 | 配置 | 说明 |
|------|------|------|
| Debug 模式 | `FLASK_DEBUG=true` | 自动重载、详细错误信息 |
| 数据库 | SQLite | 单文件，自动创建 |
| Session Cookie | `Secure=False` | HTTP 传输 |
| SECRET_KEY | 随机生成 | 每次重启变化 |
| 验证码 | 直接返回 | 不接入短信服务 |

## 3. 生产环境部署

### 3.1 服务器架构

```
│                        生产服务器                                │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Nginx (端口 80/443)                                     │   │
│  │  - 处理静态文件请求（CSS/JS/图片）                        │   │

│  │  - 反向代理到 Gunicorn                                    │   │

│                       │ Unix Socket                             │
│  ┌────────────────────┴────────────────────────────────────┐   │
│  │  Gunicorn (Unix Socket)                                  │   │
│  │  - Worker 进程数: 2-4 × CPU 核心数                       │   │
│  │  - 工作模式: sync（默认）或 gevent（WebSocket）           │   │
│  │  - 超时: 30 秒                                           │   │
│  └────────────────────┬────────────────────────────────────┘   │
│                       │                                         │

│  │  Flask Application                                       │   │
│  │  - 业务逻辑处理                                          │   │
│  │  - Session 管理                                          │   │
│  │  - WebSocket 连接                                        │   │
│  └────────────────────┬────────────────────────────────────┘   │
│                       │                                         │
│  ┌────────────────────┴────────────────────────────────────┐   │
│  │  MySQL (端口 3306) 或 SQLite                             │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Nginx 配置

```nginx
# /etc/nginx/sites-available/baoming

upstream baoming_app {
    server unix:/tmp/baoming.sock fail_timeout=0;
    # 或多实例 TCP 方式
    # server 127.0.0.1:5000;
    # server 127.0.0.1:5001;
}

server {
    listen 80;
    server_name baoming.example.com;
    
    # HTTP 跳转 HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name baoming.example.com;
    
    # SSL 证书
    ssl_certificate /etc/nginx/ssl/baoming.crt;
    ssl_certificate_key /etc/nginx/ssl/baoming.key;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    
    # 静态文件直接由 Nginx 提供
    location /static/ {
        alias /var/www/baoming/static/;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
    
    # 反向代理到 Gunicorn
    location / {
        proxy_pass http://baoming_app;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        proxy_connect_timeout 30s;
        proxy_send_timeout 30s;
        proxy_read_timeout 30s;
    }
    
    # 日志
    access_log /var/log/nginx/baoming_access.log;
    error_log /var/log/nginx/baoming_error.log;
}
```

### 3.3 Gunicorn 配置

```python
# gunicorn.conf.py

# 监听地址
bind = "unix:/tmp/baoming.sock"
# bind = "127.0.0.1:5000"

# Worker 数量：2-4 × CPU 核心数
workers = 4

# 工作模式
# sync: 默认，适合普通 HTTP
# gevent: 适合 WebSocket 长连接
worker_class = "sync"
# worker_class = "gevent"
# worker_connections = 1000

# 超时设置
timeout = 30
keepalive = 5

# 日志
accesslog = "/var/log/gunicorn/access.log"
errorlog = "/var/log/gunicorn/error.log"
loglevel = "info"

# 进程名称
proc_name = "baoming"

# 守护进程（由 systemd 管理时不需要）
daemon = False

# PID 文件
pidfile = "/tmp/baoming.pid"
```

**启动命令**：

```bash
cd /var/www/baoming
source .venv/bin/activate
gunicorn -c gunicorn.conf.py run:app
```

## 4. Docker 部署

### 4.1 Dockerfile

```dockerfile
# 书法报名管理系统 Docker 镜像
FROM python:3.10-slim

WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    gcc \
    libpq-dev \
    && rm -rf /var/lib/apt/lists/*

# 安装 Python 依赖
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 复制项目代码
COPY app/ ./app/
COPY static/ ./static/
COPY templates/ ./templates/
COPY run.py .
COPY mysql_schema.sql .

# 创建非 root 用户
RUN useradd -m -u 1000 baoming && chown -R baoming:baoming /app
USER baoming

# 暴露端口
EXPOSE 5000

# 健康检查
HEALTHCHECK --interval=30s --timeout=5s --start-period=5s --retries=3 \
    CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:5000/')" || exit 1

# 启动命令
CMD ["python", "run.py"]
```

### 4.2 Docker Compose 配置

```yaml
# docker-compose.yml
version: '3.8'

services:
  app:
    build: .
    container_name: baoming_app
    ports:
      - "5000:5000"
    environment:
      - SECRET_KEY=${SECRET_KEY}
      - ADMIN_DEFAULT_PASSWORD=${ADMIN_DEFAULT_PASSWORD}
      - USE_MYSQL=true
      - MYSQL_HOST=db
      - MYSQL_USER=baoming
      - MYSQL_PASSWORD=${DB_PASSWORD}
      - MYSQL_DATABASE=calligraphy_system
    volumes:
      - ./static:/app/static
      - ./templates:/app/templates
    depends_on:
      - db
    restart: unless-stopped
    networks:
      - baoming_network

  db:
    image: mysql:8.0
    container_name: baoming_db
    environment:
      - MYSQL_ROOT_PASSWORD=${DB_ROOT_PASSWORD}
      - MYSQL_DATABASE=calligraphy_system
      - MYSQL_USER=baoming
      - MYSQL_PASSWORD=${DB_PASSWORD}
    volumes:
      - mysql_data:/var/lib/mysql
      - ./mysql_schema.sql:/docker-entrypoint-initdb.d/init.sql
    ports:
      - "3306:3306"
    restart: unless-stopped
    networks:
      - baoming_network

  nginx:
    image: nginx:alpine
    container_name: baoming_nginx
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/conf.d/default.conf
      - ./static:/var/www/static
      - ./ssl:/etc/nginx/ssl
    depends_on:
      - app
    restart: unless-stopped
    networks:
      - baoming_network

volumes:
  mysql_data:

networks:
  baoming_network:
    driver: bridge
```

### 4.3 Docker 部署命令


## 5. 服务器配置建议

### 5.1 最低配置（≤50 用户）

| 组件 | 配置 | 说明 |
|------|------|------|
| CPU | 1 核 | 单核即可运行 |
| 内存 | 1 GB | Python + SQLite |

| 带宽 | 1 Mbps | 小型机构够用 |

| 磁盘 | 20 GB SSD | 含备份空间 |
| 带宽 | 5 Mbps | 中等并发 |
| 数据库 | MySQL 5.7+ | 独立实例或容器 |

### 5.3 高配置（500+ 用户）

| 组件 | 配置 | 说明 |
|------|------|------|
| CPU | 4 核 | 多 Worker 处理 |
| 内存 | 4 GB | 缓存 + 连接池 |
| 磁盘 | 50 GB SSD | RAID 备份 |
| 带宽 | 10 Mbps+ | 高并发支持 |
| 数据库 | MySQL 8.0 | 主从复制 |

| 系统日志 | `/var/log/syslog` | Systemd 服务日志 |

### 8.2 关键监控指标

| 指标 | 告警阈值 | 检查方式 |
|------|----------|----------|
| CPU 使用率 | > 80% | `top`, `htop` |
| 内存使用率 | > 85% | `free -m` |
| 磁盘使用率 | > 90% | `df -h` |
| MySQL 连接数 | > 80% | `SHOW STATUS LIKE 'Threads_connected'` |
| HTTP 响应时间 | > 2s | Nginx 日志分析 |
| 错误率 | > 1% | `app.log` 分析 |
