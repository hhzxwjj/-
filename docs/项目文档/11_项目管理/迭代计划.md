# 墨香书院书法课报名签到系统 - 版本迭代计划

> 当前版本：v1.0.0 | 最后更新：2026-05-22

---

## 版本规划总览

| 阶段 | 版本号 | 周期 | 目标 |
|------|--------|------|------|
| 短期 | v1.1.x | 1-2 周 | 体验打磨 + 功能补全 |
| 中期 | v1.2.x ~ v1.4.x | 1-2 月 | 核心增强 + 运营支撑 |
| 长期 | v2.0.x | 3-6 月 | 商业扩展 + 多端支持 |

---

## 一、短期迭代（v1.1.x）—— 体验打磨

**目标**：修复已知问题，优化高频交互，让系统可用、好用。

### v1.1.0 消息中心体验优化
- [x] 联系人按最新消息时间排序
- [x] 置顶联系人功能
- [x] 修复 fetch 未携带 cookie 导致"请先登录"的 bug
- [ ] 未读消息红点提示（badge 角标）
- [ ] WebSocket 推送替代 5 秒轮询（安装 simple-websocket）
- [ ] 支持发送图片/文件

### v1.1.1 课程 & 报名流程优化
- [ ] 课程支持多时段排期（而非单一开始/结束日期）
- [ ] 报名后生成电子报名确认单（PDF 或页面）
- [ ] 课程名额满员后自动显示"候补报名"
- [ ] 课程详情页增加"课程大纲"富文本编辑

### v1.1.2 视觉 & 交互细节
- [ ] 全局添加骨架屏（skeleton）加载状态
- [ ] 移动端响应式适配（当前侧边栏在手机上体验差）
- [ ] 表单输入添加防抖和实时校验
- [ ]  toast 提示统一使用更精致的动画

---

## 二、中期迭代（v1.2.x ~ v1.4.x）—— 核心增强

**目标**：构建完整的教务运营闭环，从"能用"到"好用"。

### v1.2.0 考勤体系升级
- [ ] 支持二维码扫码签到（学员端展示二维码，管理员扫码）
- [ ] 请假审批流程（家长申请 → 管理员审批）
- [ ] 考勤统计报表（按学员/按课程/按月导出 Excel）
- [ ] 缺勤自动提醒（消息中心推送缺勤通知）

### v1.2.1 课时 & 财务模块
- [ ] 课时包购买（一次性购买多课时，按次扣减）
- [ ] 缴费记录与电子收据
- [ ] 学员课时预警（剩余课时 < 3 时自动提醒）
- [ ] 简易财务报表（收入统计、课程营收排行）

### v1.3.0 学员作品 & 成长档案
- [ ] 学员作品上传（每次课后提交练习作品）
- [ ] 作品点评（管理员文字+评分点评）
- [ ] 个人成长时间轴（报名 → 每次作品 → 结业）
- [ ] 作品墙 / 画廊展示（landing 页展示优秀学员作品）

### v1.3.1 通知 & 营销体系
- [ ] 系统公告（管理员发布，所有用户可见）
- [ ] 课程开课提醒（开课前 1 天自动消息推送）
- [ ] 新用户注册欢迎流程（引导完善孩子信息）
- [ ] 优惠券/体验课功能（限时免费试听券）

### v1.4.0 数据 & 权限
- [ ] 数据导出（学员名单、签到记录、财务流水全部支持 Excel 导出）
- [ ] 操作日志（管理员关键操作留痕）
- [ ] 多管理员角色细分（超级管理员 / 普通管理员 / 教师）
- [ ] 数据备份与恢复（数据库一键导出）

---

## 三、长期迭代（v2.0.x）—— 商业扩展

**目标**：产品化，支持规模化运营和多端使用。

### v2.0.0 多端支持
- [ ] **小程序端**（家长端：报名、签到、查课时、看作品、消息沟通）
- [ ] **小程序端**（教师端：课堂点名、发布作业、点评作品）
- [ ] H5 分享页（课程分享、作品分享带海报生成）

### v2.0.1 品牌 & 运营
- [ ] 自定义主题色（机构可自行配置品牌色）
- [ ] 自定义 landing 页内容（轮播图、文案、课程模块拖拽配置）
- [ ] SEO 优化（landing 页生成静态 HTML，利于搜索引擎收录）
- [ ] 在线客服机器人（常见问题自动回复）

### v2.1.0 SaaS 化探索
- [ ] 多机构入驻（一个系统支持多个书法培训机构）
- [ ] 机构独立域名/子域名
- [ ] 套餐计费（按学员人数或功能模块收费）

## 四、技术债务 & 基建（贯穿各版本）

| 优先级 | 事项 | 建议版本 |
|--------|------|---------|
| 🔴 高 | 数据库迁移工具（Alembic / Flask-Migrate） | v1.2.0 |
| 🔴 高 | 单元测试覆盖率提升到 60%+ | v1.2.0 |
| 🟡 中 | 生产环境部署文档（Docker + Nginx + Gunicorn） | v1.2.0 |
| 🟡 中 | API 接口统一 RESTful 规范 + 文档（Swagger） | v1.3.0 |
| 🟢 低 | 前端组件化（提取可复用组件，减少模板重复） | v1.4.0 |
| 🟢 低 | 性能优化（Redis 缓存热点数据、数据库索引优化） | v2.0.0 |

---

## 五、各版本发布 Checklist

每次发版前确认：

- [ ] 代码通过全部单元测试
- [ ] 数据库迁移脚本已测试（如有 schema 变更）
- [ ] `.gitignore` 已排除敏感文件（db、log、env）
- [ ] `static/style.css` 版本号已递增（`?v=N`）
- [ ] 更新 `CHANGELOG.md`
- [ ] 打 Git Tag（如 `git tag v1.1.0`）
